# RunHop Web

This is the React frontend for RunHop.

It currently contains:

- a marketing landing page
- login and registration routes
- a small auth API helper layer
- frontend-specific learning docs in `docs/`

## Stack

- React 18
- Vite
- React Router DOM
- plain CSS

## Scripts

From `apps/web`:

```bash
npm run dev
npm run build
npm run preview
npm run docs
```
